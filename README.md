
# CzechSimpleCover5e

Czech translation of Simple Cover 5e version 1.4.4

## Popis
Tento modul poskytuje českou lokalizaci pro Simple Cover 5e verze 1.4.4. Obsahuje překlady základních textů aplikace.

## Instalace
Pro instalaci modulu obvyklým upůsobem můžeš použít následující odkaz ve Foundry VTT:

```
https://raw.githubusercontent.com/Lethrendis/CzechTranslationSimpleCover5e/main/module.json
```
